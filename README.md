## Installation

To set up this portfolio locally, follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/Abhik-Develops/portfolio.git
    ```

2. Navigate to the project directory:

    ```bash
    cd portfolio
    ```
3. Install required packages:

    ```bash
    npm install
    ```

4. Run command:

    ```bash
    npm run dev
    ```
